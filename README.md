# Artificial-Neural-Network-Example
Running python 3.7.4 64-bit with TensorFlow 2.4.1

Builds, trains, and evaluates a 1-6-1 sequential model neural network using the [Qualitative Bankruptcy](https://archive.ics.uci.edu/ml/datasets/qualitative_bankruptcy) data set from UCI Machine Learning Repository.
